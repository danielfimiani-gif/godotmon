Question <bdi></bdi>
Hacer un videojuego en 2D o 3D de su autoría completamente original.

Se deberán cumplir ciertas condiciones para estar en un determinado "tier":



Bronce (1 - 4):

Splash Screen, Main Menu, Game Scene y Option Scene.
Condición de victoria y derrota.
Dificultad progresiva.
UI animada.
Arte y sonido.
Pagina de Itch.io (base).
Pitch.


Plata (5 - 8):

Game juice (partículas y screen shake).
Level Selector.
Pagina de itch.io con screenshots y descripción del juego.


Oro (8 - 10):

Mecánica innovadora en el género.


Nota: Recuerden que TODO lo que le AGREGUEN EXTRA, SUMA para USTEDES. Así que no se queden solo con esto y hagan mucho mas.
